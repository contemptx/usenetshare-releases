# UsenetShare — Releases

Official desktop app releases for **UsenetShare**: installers and the
Ed25519-signed update manifest (`latest.json`) the app uses to check for and
verify updates.

This repository contains **binaries only — no source code**.

## Downloads

Grab the newest installer from the [latest release](../../releases/latest):

- **Windows:** `UsenetShare-windows-x86_64-Setup.exe`
- **Linux:** `UsenetShare-linux-x86_64.tar.gz`

The app can also update itself: **Settings -> Updates -> Check for updates**.
Every update is verified (Ed25519-signed manifest + SHA-256) before it is
applied.

## Support

Report a bug or get help from **Settings/Help** inside the app, or email
**support@usenetshare.com**.
